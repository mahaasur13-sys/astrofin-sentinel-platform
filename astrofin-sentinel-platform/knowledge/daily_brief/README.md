# Daily Brief — Интеграция внешних агентов

**Назначение:** Хранение и обработка ежедневных email-сводок от внешнего агента (R-040).

**Формат файлов:**
- `brief_YYYY-MM-DD.md` — сводка за день
- `brief_latest.md` — symlink на последнюю сводку
- `ideas.md` — сгенерированные идеи для ATOM

**TTL:** Хранить 30 дней, старые удаляются автоматически.

**CLI:** `python tools/thompson_cli.py daily-brief`
