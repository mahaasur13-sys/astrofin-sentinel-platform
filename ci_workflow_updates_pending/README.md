# Pending CI workflow updates (manual apply)

Эти обновлённые workflow-файлы пришли из более новой вложенной копии в ходе
консолидации (Вариант C), но **не могут быть запушены GitHub App'ом** без права
`workflows`. Поэтому они вынесены сюда и НЕ применены к `.github/workflows/`.

| Файл | Тип |
|------|-----|
| `ci.yml` | обновление (крупный рефакторинг CI) |
| `ci.security.yml` | обновление (минорное) |
| `pr-checks.yml` | обновление (минорное) |
| `daily-calibration.yml` | **новый** workflow |

## Как применить
Один из вариантов:
1. **Выдать право** GitHub App `workflows` (Settings → GitHub Apps) и затем
   скопировать эти файлы в `.github/workflows/`, закоммитить и запушить.
2. **Вручную** через веб-интерфейс GitHub скопировать содержимое в
   `.github/workflows/` (у владельца репозитория есть права на workflow).

После применения эту папку можно удалить.
