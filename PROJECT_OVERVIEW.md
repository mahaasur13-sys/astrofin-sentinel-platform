# AstroFin Sentinel V5 — Обзор проекта

**AstroFin Sentinel V5** — мультиагентная система генерации торговых сигналов, объединяющая 20 доменных специалистов (технический, макроэкономический, астро‑финансовый, сентимент, фундаментальный, количественный, опционный анализ) с мета-RL эволюцией стратегий (генетический алгоритм + KARL Q‑обучение), семантическим поиском RAG на базе FAISS и синтезом взвешенного консенсуса совета агентов.

## Ключевые возможности
- **20 специализированных агентов** в 4 пулах (Technical, Macro, Astro, Sentiment).
- **RAG-поиск** по астрологическим, техническим и макроэкономическим знаниям (FAISS + Ollama nomic-embed-text).
- **Совет директоров** с Thompson Sampling для динамического выбора агентов.
- **Синтез KARL** — 13-шаговый RL-пайплайн агрегации сигналов с Q-обучением.
- **Мета-RL эволюция** стратегий (генетический алгоритм + walk-forward валидация).
- **Risk Engine V2** с kill-switch, лимитами экспозиции и корреляций.
- **GitAgent Export** — стандартизированные пакеты агентов для деплоя.
- **Data Room** — архитектура прозрачности данных (инвентаризация, журнал конфликтов).
- **Мониторинг** Prometheus + Grafana с доменными алертами.

## Доменная структура (DDD)
Проект организован вокруг следующих ключевых доменов:
1. **Market Data** — сбор, очистка, кэширование рыночных данных.
2. **Signal Synthesis** — генерация сигналов агентами, синтез KARL.
3. **Risk Management** — контроль рисков, kill-switch, Safety Gate.
4. **Execution** — отправка ордеров, следование сигналам (paper/live trading).
5. **Evolution** — оптимизация стратегий через Meta-RL.
6. **Observability** — метрики, алерты, дашборды.
