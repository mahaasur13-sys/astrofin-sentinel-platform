# Критерии успеха

| Метрика | Целевое значение |
|---------|------------------|
| Покрытие тестами | >80% (meta_rl, trading, agents) |
| Задержка синтеза (p95) | <500 мс |
| Стабильность | 0 CRITICAL алертов за 7 дней |

| Экспорт стратегий | Детерминированный, без ошибок валидации |
| Кэш‑хитрейт | >70% для рыночных данных |
| Алерты | Настроены и проверены: KillSwitch, AgentErrorRate, SynthesisLatency |
